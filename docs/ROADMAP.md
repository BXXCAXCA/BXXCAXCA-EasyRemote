# Roadmap

## Phase 0: Scaffold

- Flutter shell
- Demo device cards
- Domain models
- Placeholder services
- Product and security documents

## Phase 1: Local inventory

- SQLite persistence
- Device create/edit/delete
- Grouping and favorites
- Import and export

## Phase 2: Sync and vault

- WebDAV configuration sync
- Sensitive-field encryption
- Platform keystore integration

## Phase 3: Adapter registry

- Declarative adapter metadata
- User-installed tool detection
- Status display
- Route merge and scoring

## Phase 4: Protocol entry points

- Authorized protocol launchers
- User confirmation flows
- Connection history
- Diagnostics

## Phase 5: Native core research

- Platform service boundary
- Optional self-hosted service support
- Remote-session research boundary
