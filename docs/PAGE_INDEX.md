## Page guidance

Pages should stay small and delegate logic to services or controllers.

When adding a new page:

1. Add the feature page under `lib/src/features/`.
2. Add focused services or models first.
3. Add a smoke test or service-level test.
4. Update this index if the page is user-facing.
