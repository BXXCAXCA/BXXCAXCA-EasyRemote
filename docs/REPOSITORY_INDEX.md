# Repository Index

## App

- `lib/main.dart` starts the Flutter app.
- `lib/src/app/` contains app bootstrap and shell wiring.
- `lib/src/features/` contains UI pages.
- `lib/src/models/` contains data models.
- `lib/src/services/` contains planning, validation, and boundary services.
- `lib/src/repositories/` contains local repository abstractions.
- `lib/src/state/` contains UI controllers.
